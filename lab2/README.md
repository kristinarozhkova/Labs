  - Task 5: Динамічний GET-запит на курс валют

  Для отримання курсу валют буде використано офіційне API Національного банку України, згідно з наданою специфікацією. Було реалізовано серверний скрипт, який обробляє GET-запит за маршрутом /currency із параметрами today або yesterday. Залежно від отриманого параметра сервер формує відповідний запит до API НБУ та повертає курс чеської крони (CZK), актуальний на поточну або попередню дату.

Після запуску серверу та виконання відповідного запиту через браузер, було отримано правильне значення курсу, що підтверджує коректну роботу реалізованого динамічного обробника. На рисунку 5 та 6 відображено результат успішного запиту, де сервер повертає курс чеської крони згідно з даними НБУ.

  - Task 6: Обробка методу POST зі збереженням в файлі

  Для реалізації цього завдання сервер було налаштовано таким чином, щоб приймати POST-запити, у тілі яких передаються текстові дані. Отриманий текст зберігається у файл data.txt на сервері. Для цього в коді був створений окремий маршрут /save, який обробляє POST-запити, зчитує їхній вміст і додає його у файл.
  Після запуску серверу та виконання POST-запиту через термінал було передано текстові дані, які успішно збережено у файлі. На рисунку 6 показано використану команду curl, за допомогою якої було здійснено відправку текстового повідомлення на сервер.
