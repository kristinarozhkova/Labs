Лабораторна робота 3: Проектування REST API 
---------------------------------------------------
  1. Розробка REST API веб-сервісу з використанням flask

&nbsp;&nbsp;&nbsp;&nbsp;Під час виконання лабораторної роботи було розроблено REST API веб-сервіс із використанням фреймворку Flask. У процесі реалізації було дотримано всіх вимог, зазначених у завданні. Зокрема, у веб-сервісі створено два необхідні ендпоінти: /items, який відповідає за роботу з усім каталогом товарів, та /items/<id>, що дозволяє отримувати, оновлювати або видаляти інформацію про конкретний товар за його ідентифікатором.
  Також у роботі реалізовано підтримку чотирьох основних HTTP-методів: GET для зчитування даних, POST для створення нового запису (доступний тільки для ендпоінту /items), PUT для оновлення товару та DELETE для його видалення. Всі запити й відповіді обробляються у форматі JSON, що відповідає вимогам до побудови REST API.
  Окремо було додано HTTP Basic Authentication. Дані користувачів не зберігаються в коді, а зчитуються з окремого файлу users.txt, що відповідає середньому рівню складності (Medium). Аналогічно, каталог товарів теж зберігається у зовнішньому файлі items.json, куди сервіс записує всі створені, змінені або видалені записи.
  Увесь програмний код веб-сервісу наведено у файлі script, а також у проєкт додано відповідні файли users.txt і items.json, необхідні для коректної роботи програми.
